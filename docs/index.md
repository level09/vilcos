## Purpose

This guide aims to provide instructions on installing and utilizing the Vilcos framework, a modern Flask-based web framework with Supabase integration.

## Synopsis

Vilcos is a Flask application template featuring pre-configured modules and pre-written boilerplate code, designed to significantly reduce the development time for your upcoming project. It leverages Supabase for backend services, providing a robust and scalable foundation for web applications.

## What's included?

* Flask-based architecture with blueprints
* Supabase integration for authentication and backend services
* Vue3 and Vuetify3 integration for frontend development
* Axios for HTTP requests
* Customizable configuration system
* Easy-to-use CLI commands for common tasks

## Getting Started

1. Clone the Vilcos repository
2. Set up your virtual environment and install dependencies
3. Configure your Supabase credentials in the `.env` file
4. Run the development server using `flask run`

For detailed installation and setup instructions, please refer to the README.md file in the project repository.

## Supabase Integration

Vilcos uses Supabase for authentication and backend services. This integration provides:

* User registration and login
* Session management
* Password reset functionality
* Access to Supabase's powerful database and storage features

To learn more about how to use Supabase in your Vilcos project, check out the Supabase documentation and the Vilcos-specific guides in this documentation.
